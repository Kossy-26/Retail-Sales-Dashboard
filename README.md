# Canadian Retail Sales Dashboard

## What This Project Is About
This project analyzes 31 years of Canadian retail sales data
(January 1991 to December 2022) from Statistics Canada.
The goal was to clean the raw data using Power Query, build
pivot table summaries, and create a one-page Excel dashboard
that communicates key business findings clearly.

## What I Did
- Imported 142,000 rows of raw CSV data into Excel
- Used Power Query to remove unnecessary columns, filter to
  Unadjusted data only, remove city-level rows, fix date
  formatting, and clean the retail category column
- Built 3 pivot tables: sales by province, monthly sales
  trend, and sales by retail category
- Created a line chart, bar chart, and pie chart
- Designed a one-page dashboard with KPI boxes and key findings

## Tools Used
- Microsoft Excel (Power Query, Pivot Tables, Charts, Dashboard)

## Key Findings
- Ontario and Quebec account for the majority of Canadian retail sales
- Motor Vehicles & Parts Dealers is the largest retail category
- Sales dropped sharply in early 2020 (COVID-19) and recovered by late 2020
- Cannabis stores appear from 2018 onward following legalization

## Data Source
